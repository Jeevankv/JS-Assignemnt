Datatypes in Javascript

Datatypes:
Number
String :string is sequence of character
Boolean : Booleans can only have two values: true or false
Array : arrays are written with square brackets, array items are separated by commas
Object : Object properties are written as name:value pairs, separated by commas
Undefined : variable without a value, has the value undefined
Null : null is "nothing". It is supposed to be something that doesn't exist
